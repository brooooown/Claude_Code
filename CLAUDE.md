# Claude Code プロジェクト設定

## タスク完了時の通知
- タスクが完了した際は、macOS通知を出してユーザーに知らせる
- `osascript -e 'display notification "タスクが完了しました" with title "Claude Code"'` を使用
- 重要なタスクや長時間実行タスクの完了時は必ず通知する

## Project Rules

### Task Completion Notifications
- Send macOS notifications when tasks are completed
- Use `osascript -e 'display notification "Task completed" with title "Claude Code"'`
- Always notify for important or long-running tasks

### General Guidelines
- このプロジェクトではユーザーの作業効率を重視する
- タスクの進捗と完了状況を明確に伝える
- 日本語と英語の両方でサポートする
