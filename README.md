# iOS App Prototype

iOS風のアプリプロトタイプを実装したWebアプリケーションです。

## 🚀 デモ

**GitHub Pages:** https://y-nagakura.github.io/Claude_Code/

パスワード: `mirai`

## ✨ 特徴

### 🎨 リキッドグラス・デザイン
- 本格的なガラスモーフィズム効果
- `backdrop-filter: blur(48px) saturate(220%) brightness(1.08)`
- 多層グラデーションとスペキュラーハイライト

### 📱 マルチスクリーン設計
- **ホーム画面**: メルカリ風メイン画面
- **検索画面**: フローティング検索バー付き
- **リスト画面**: いいね・最近見た商品
- **オークション画面**: 注目商品カルーセル
- **ウォレット画面**: メルペイカード・取引履歴
- **マイページ**: サイドスライドオーバーレイ

### 🎯 インタラクション
- ハプティックフィードバック
- スムーズなタブ切り替え
- キーボード対応（visualViewport API）
- Safe Area対応（iPhone X以降）

### 📐 レスポンシブ対応
- モバイルファースト
- iPhone各サイズに最適化
- ダイナミックビューポート（100dvh）

## 🛠 技術スタック

- **HTML5** - セマンティック構造
- **CSS3** - ガラスモーフィズム、Grid、Flexbox
- **JavaScript** - バニラJS（フレームワークレス）

## 💾 主要ファイル

- `index.html` - メインプロトタイプ（元：05.html）
- `original-05.html` - オリジナルファイル
- `liquid-nav.*` - リキッドグラス・ナビゲーション
- `ios-app-prototype.*` - 分割版
- `mercari-prototype.html` - メルカリ風プロトタイプ

## 🎮 使用方法

1. ページを開く
2. パスワード「mirai」を入力
3. 各タブをタップして画面切り替えを体験
4. プロフィールボタンでマイページオーバーレイを開く
5. 検索画面でキーボードインタラクションを確認

## 🔐 認証

- パスワード認証システム搭載
- セッションストレージで認証状態管理
- 認証キー: `auth_05`

## 📱 対応デバイス

- iPhone (Safari)
- Android Chrome
- デスクトップブラウザ（プレビュー用）
