# Claude UX Workspace

UXデザインプロセスをサポートするワークスペースです。

## 構造

```
claude-ux-workspace/
├── idea-capture/          # Raw insights, frameworks
├── document-generation/   # Strategy docs, alignment materials
├── stakeholder-tools/     # Communication templates
├── projects/mercari/      # Project-specific work
├── visual-frameworks/     # Process flows, diagrams
└── templates/             # Reusable formats
```

## ディレクトリ詳細

- **idea-capture**: アイデアや洞察、フレームワークの生のデータ
- **document-generation**: 戦略ドキュメント、調整用資料
- **stakeholder-tools**: コミュニケーションテンプレート
- **projects/mercari**: プロジェクト固有の作業内容
- **visual-frameworks**: プロセスフロー、ダイアグラム
- **templates**: 再利用可能なフォーマット
